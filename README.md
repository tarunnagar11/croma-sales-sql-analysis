# 🖥️ Croma Electronics Store Analysis using SQL

## 📌 Overview

This project analyzes Croma's product inventory and manufacturer data using MySQL. It demonstrates SQL skills by solving business-oriented queries related to product pricing, manufacturer performance, inventory management, and sales insights.

---

## 🛠️ Technology Used

- MySQL

---

## 📂 Dataset

The project uses two datasets:

- `product_inventory.csv`
- `manufacturers.csv`

Both datasets are available in the **Dataset** folder.
